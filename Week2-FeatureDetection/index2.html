HTML Features w. fallback: Media Capture

<input type="file" accept="image/*" capture="camera" > will fallback to <input type="file"> on unsupported browsers.

<input type="file" accept="audio/*" id="capture" capture="microphone" > will fallback to <input type="file"> on unsupported browsers.

<input type="file" accept="image/*" capture="camera" > will fallback to <input type="file"> on unsupported browsers.
