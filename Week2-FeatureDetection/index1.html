HTML Features w. fallback: inputs

<input type="search"> will fallback to <input type="text"> on unsupported browsers.
Will have a placeholder 'Search' telling the user the expected input type.

<input type="color"> will fallback to <input type="text"> on unsupported browsers.
Will have a placeholder 'Color' telling the user the expected input type.

<input type="time"> will fallback to <input type="text"> on unsupported browsers.
Will have a placeholder 'Time: hh-mm-ss' telling the user the expected input type.
